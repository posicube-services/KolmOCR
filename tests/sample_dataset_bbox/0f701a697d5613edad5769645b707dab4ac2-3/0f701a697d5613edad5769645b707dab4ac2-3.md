<!-- bbox: [28,58,438,35] -->
우리는 대체를 수행합니다: \$s = \frac{1 - r^n}{1 + r^n} \$, 그리고 결과적인 적분

<!-- bbox: [28,114,419,67] -->
\[
4 \int_0^1 (e^{-as} - e^{-a/s})(1 - s)^{-1 + 1/n}(1 + s)^{-1 - 1/n} ds
\]

<!-- bbox: [28,203,972,34] -->
에서 \( (0, a^{1/2}) \) 구간 이외의 모든 것을 제외합니다. 그러면 대수적 인자들을 상수로 대체할 수 있으며, 결정해야 할 양은 다음보다 큽니다:

<!-- bbox: [28,258,218,71] -->
\[
K_3 \int_0^{a^{1/2}} (e^{-a} - e^{-a/s}) ds.
\]

<!-- bbox: [28,350,832,34] -->
각 구간 \([(j-1)a, ja]\) $j = 1, 2, \(\cdots\$, \$[a^{-1/2}]\$)를 따로 고려합니다. j번째 구간에서 적분의 최솟값은 다음과 같습니다:

<!-- bbox: [28,406,266,31] -->
\[
e^{-a} - e^{-a/j} > -a + j^{-1} - j^{-2},
\]

<!-- bbox: [28,458,265,34] -->
따라서 적분의 값은 다음보다 큽니다:

<!-- bbox: [28,514,327,89] -->
\[
a \sum_{j=1}^{[a^{-1/2}]} [-a + j^{-1} - j^{-2}] > K_4 a |\log a|.
\]

<!-- bbox: [28,624,438,34] -->
이제 \$k = 2, 3, \cdots \$에 대해 \$a_k = k^{-1}(\log k$^{-3/2} \)라고 하고,

<!-- bbox: [28,680,283,65] -->
\[
f(z) = \exp \left$- \sum a_k \frac{1 + z^n k}{1 - z^n k} \right$.
\]

<!-- bbox: [28,766,607,34] -->
만약 \$n_k \to \infty \$가 충분히 빠르게 이루어진다면, \$f \$는 다시 원하는 성질을 갖게 됩니다.
